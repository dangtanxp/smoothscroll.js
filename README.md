smoothscroll.js
===============

Smooth Scroll all website replace scroll default use jQuery

##EMBED TO WEBSITE

```html
<html>
<head>
<style>
body{
	height:5000px;
}
</style>
<script src="https://code.jquery.com/jquery-2.1.1.min.js"></script>
<script src="smoothscroll.min.js"></script>

```
